# OP Gear Minecraft Paper Server Mod

Plugin for [Paper](https://papermc.io/software/paper/) 1.21.11 som legger til OP-utstyr (våpen og rustning) som **kun operatorer kan se og bruke**.

---

## Nedlasting

**Mod (plugin):** [op-gear-minecraft-paper-server-mod.jar](https://github.com/matwebmaker-lab/Mod-for-my-minecraft-server/raw/main/op-gear-minecraft-paper-server-mod.jar)

Last ned JAR-filen og legg den i Paper-serverens `plugins/`-mappe. Start serveren på nytt.

---

## Installasjon

1. Last ned **op-gear-minecraft-paper-server-mod.jar** (lenke over).
2. Legg filen i `plugins/`-mappen på Paper-serveren (Minecraft 1.21.11).
3. Start eller restart serveren.
4. Gjør spillere OP (f.eks. `op Brukernavn` i konsollen) – da får de tilgang til OP-utstyret.

---

## Kommandoer

| Kommando | Beskrivelse |
|----------|-------------|
| `/lager` | Viser liste over alle OP-items |
| `/lager give <id>` | Gir deg OP-item (kun for OP) |
| `/lagerkiste` | Åpner admin-kisten med OP-utstyr (kun Admin_owner) |

---

## OP-rustning (effekter når du har den på)

| Rustning | Effekt |
|----------|--------|
| **Hjelm** | +20 ekstra liv |
| **Brystplade** | 10 blokker reach |
| **Bukse** | Styrke II + uendelig vanndråper |
| **Støvler** | Speed II |
